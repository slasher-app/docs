# Slasher Documentation

Official website and documentation for [Slasher](https://github.com/slasher-app/slasher) - slash commands everywhere!

## Development

Install dependencies:
```bash
bun install
```

Start the dev server:
```bash
bun run dev
```

Build for production:
```bash
bun run build
```

## Tech Stack

- Next.js 16 with Nextra
- Tailwind CSS 4
- Pagefind for search
