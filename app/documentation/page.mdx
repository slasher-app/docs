## Installation

### Chrome

To install Slasher in Chrome (or derivatives like Edge and Arc), simply visit the [Chrome Web Store]() and click the "Add to Chrome" button. Once installed, you can access Slasher by clicking the extension icon in your browser toolbar.

### Firefox

To install Slasher in Firefox, simply visit the [Firefox Add-ons]() and click the "Add to Firefox" button. Once installed, you can access Slasher by clicking the extension icon in your browser toolbar.
